# Cómo agregar un nuevo curso

1. **Subir el certificado** a la carpeta correspondiente (PDF o imagen).
   - Nombrar el archivo: `nombre-del-curso_institucion_año.pdf`
   - Ejemplo: `niif-pymes_icpa_2024.pdf`

2. **Editar el README de la categoría** y agregar una fila a la tabla.

3. **Actualizar el README principal**:
   - Agregar fila en la sección correspondiente.
   - Sumar 1 al contador de la categoría en el resumen.
   - Actualizar la fecha de última actualización.
